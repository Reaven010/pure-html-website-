  2. File Path

    file path plays vital role in linking webpages together and also linking images/videos to web page
    there are two types of paths

    • absolute path : absolute path is the complete address of a file from a system which access complete pc to get the file
      which is very dangerous and can cause security issues if not carefull hence by default web browsers block uses of absolute paths
    • relative path : this is the path of the file from current location only which can be hosted and hence is relatively safe aas going outside of bounds is not allowed

  5. Class

    • script

  6. ID
